# MVP 0.4 — Cartas Básicas e Interface de Decisão

Este diretório consolida o escopo documental do **MVP 0.4** do Monstrinhomon.

## Objetivo

Definir, sem alterar runtime, o plano mínimo para introduzir cartas no Wild Loop com baixo risco.

## Documentos

- `MVP_0_4_CARTAS_BASICAS.md`: escopo funcional/técnico das cartas básicas.
- `BACKLOG_MVP_0_4.md`: backlog priorizado para implementação incremental.
- `CRITERIOS_DE_PRONTO_MVP_0_4.md`: Definition of Ready/Done e gates de qualidade.
- `PROMPT_IMPLEMENTACAO_CARTAS_BASICAS.md`: prompt operacional para próximo PR de implementação pequeno e seguro.
- `PRE_MVP_0_4_ACTION_PLAN.md`: base histórica da estabilização pré-MVP 0.4.

## Limite desta entrega

- **Sem mudanças de runtime**.
- **Sem implementação de cartas**.
- **Sem alteração de save/data/combat core**.

Esta pasta prepara o terreno para um PR seguinte de execução técnica controlada.
